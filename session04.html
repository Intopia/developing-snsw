Quick recap of video material

Inputs

  • Use semantic form elements
  • Use the autocomplete attribute
  • >Use the inputmode attribute

Labels

  • Programmatically associate labels with form controls
  • Be careful when hiding labels

Instructions

  • How to define fields as "required"
  • How to programatically associate hint text

Errors

  • How to programatically associate error messages
  • Focus management to fields in error
  • What to do with mutiple errors

Notifications

  • Timing - let users decide when to close
  • Proximity - be aware of users such as Zoomtext

Summary

  • Make forms as simple as possible for users to fill out
  • Provide clear and concise labels, instructions, and errors
  • Make sure that focus is managed correctly when errors do occur
  • Ensure that all users are able to access and read notification
