Exercise: Accessible SVGs

Accessing the exercise:

DEVELOPER EXERCISE: Accessible SVGs.

Add a title

  1. Add a <title> element as the first child of the <svg> element.
  2. Give the <title> element an id attribute with a value of "aaa".

Add a description

  1. Add a <desc> element after the <title> element
  2. Give the <desc> element an id attribute with a value of "bbb".

Add a role to the svg

  1. Add role="img" to the <svg> element.

Link the title and desc to the SVG

  1. Give the <svg> element an aria-labelledby attribute with a value of "aaa".
  2. Give the <svg> element an aria-describedby attribute with a value of "bbb".

Example 2

  1. Set role="none".
  2. Set focusable="false".
  3. Add a label to the button.
